Introduction

Python is a high-level, interpreted programming language that has gained immense popularity due to its simplicity and readability. It was created by Guido van Rossum and was first released in 1991. Python is known for its clean and easy-to-understand syntax, which makes it an ideal choice for beginners while also offering powerful tools for experienced developers.

Python is an open-source language, meaning its source code is freely available to the public. This has helped it evolve rapidly with a vast community contributing to its growth. Over the years, Python has become one of the most widely used languages in the world, powering everything from web development and data analysis to machine learning and artificial intelligence.

One of Python's standout features is its support for multiple programming paradigms. It can be used for object-oriented programming (OOP), procedural programming, and even functional programming. This versatility allows developers to choose the best approach for the task at hand, which is one of the reasons Python is so popular in various fields, including software engineering, web development, scientific computing, and automation.

Data Types

Python, as a dynamically typed language, allows variables to hold values of different types without the need to explicitly declare their type. Python provides several built-in data types to support various kinds of operations. The most commonly used data types in Python include integers, floats, strings, lists, tuples, sets, and dictionaries.

1. Integers (int)

Integers are whole numbers, which can be either positive or negative, and can range from small to very large values, depending on the system's architecture. Python provides built-in support for large integers, which makes it stand out in handling computations that involve big numbers.

x = 5
y = -10

2. Floating-Point Numbers (float)

Floating-point numbers, or simply floats, represent real numbers and are written with a decimal point. They are used to handle calculations involving decimals, such as scientific calculations or precise measurements. In Python, you can perform operations with floats and even use them for division.

pi = 3.14159
temperature = -12.5

3. Strings (str)

Strings are sequences of characters enclosed in single, double, or triple quotes. Python strings are immutable, meaning their content cannot be changed after they are created. They are commonly used to store text and can be manipulated with various string methods, such as concatenation, slicing, and formatting.

name = "John Doe"
greeting = 'Hello, World!'

4. Lists (list)

Lists are ordered collections of items, which can be of any data type. Lists are mutable, meaning their elements can be changed after creation. Lists are denoted by square brackets [] and can hold a mix of data types, such as numbers, strings, or other lists. They support indexing, slicing, and many useful methods for data manipulation.

fruits = ['apple', 'banana', 'cherry']
numbers = [1, 2, 3, 4, 5]
mixed = ['apple', 3, 4.5, True]

5. Tuples (tuple)

Tuples are similar to lists but are immutable, meaning their values cannot be changed once created. Tuples are denoted by parentheses (). They are used to store fixed collections of items, which can be of different data types. Tuples are useful when you want to ensure the data remains constant throughout the program.

coordinates = (10, 20)
person = ('John', 25, 'Engineer')

6. Sets (set)

Sets are unordered collections of unique elements. Sets are mutable, meaning you can add or remove elements after their creation, but they do not allow duplicate values. Sets are denoted by curly braces {} and are commonly used for operations like union, intersection, and difference. They are great for storing items where duplicates are not allowed.

colors = {'red', 'blue', 'green'}
unique_numbers = {1, 2, 3, 4, 5}

7. Dictionaries (dict)

Dictionaries are unordered collections of key-value pairs. Each key in a dictionary is unique, and each key maps to a specific value. Dictionaries are mutable and are denoted by curly braces {}, with a colon : separating keys and values. Dictionaries are highly versatile and often used for storing and retrieving data in the form of key-value relationships.

person_info = {'name': 'John', 'age': 25, 'profession': 'Engineer'}
contacts = {'John': '123-456-7890', 'Jane': '987-654-3210'}

8. Booleans (bool)

Booleans are a data type with only two possible values: True or False. They are commonly used for conditions and logical operations. In Python, the boolean values are of type bool.

is_active = True
is_eligible = False

9. None Type (NoneType)

The None type is a special type used to represent the absence of a value or a null value. It is commonly used for default values in function arguments or as a placeholder when a variable or object is yet to be assigned a meaningful value.

nothing = None

Advanced Topics

Python also supports advanced features such as decorators, metaclasses, and concurrency.


def decorator_function(original_function):
    def wrapper_function():
        print("Wrapper executed before {}".format(original_function.__name__))
        return original_function()
    return wrapper_function

@decorator_function
def display():
    print("Display function ran")

display()

Testing in Python

Python provides the unittest framework for writing test cases.

class TestMath(unittest.TestCase):
    def test_addition(self):
        self.assertEqual(2 + 2, 4)

if __name__ == '__main__':
    unittest.main()
